# Interactive World Countries by Land Area

This project is an **interactive HTML visualization** of all countries in the world by land area. Users can view countries ranked by size and search for specific countries to focus on their data.

## Features

- **Interactive bar chart** showing countries by land area (square miles)
- **Search functionality** to filter countries dynamically
- Fully **responsive** and works in any modern web browser

## How to Use

1. **Download or clone** this repository.
2. Open the `index.html` file in a modern browser (Chrome, Firefox, Edge, or Safari).
3. The chart will display countries in **descending order of land area**.
4. Use the **search box** to quickly filter and highlight specific countries.

## Technologies Used

- **HTML & CSS** – Page structure and styling
- **JavaScript** – Adds interactivity and search functionality
- **Chart.js** – Library used for rendering the bar chart
